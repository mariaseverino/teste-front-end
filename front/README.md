## 🧰 Stack utilizada

- Vite
- TypeScript
- React
- Sass

Instale com npm

```bash
# Clone o repositório
git clone https://github.com/mariaseverino/teste-front-end

# Acesse o diretório do projeto
cd teste-front-end/front

# Instale as dependências
npm install

# Inicie o projeto
npm run dev
```
